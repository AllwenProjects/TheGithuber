#Installl
<b> اموزش نصب</b>
```sh
sudo apt-get update
git clone https://github.com/taylor-team/github-bot
cd github-bot
sudo apt-get install python-setuptools
easy_install pyTelegramBotAPI
easy_install urllib
# Add Token file Main.py line 9 " Your Token "
python main.py
```
